# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately via [GitHub Security Advisories](https://github.com/ofershap/cursor-plan-preview/security/advisories/new).

Do not open a public issue for security vulnerabilities.

## Scope

This project runs locally and shares data via URL hash encoding (client-side only). No data is sent to external servers. The local HTTP server binds to `127.0.0.1` and is not accessible from the network.

## Response

I'll acknowledge reports within 48 hours and aim to release a fix within 7 days for confirmed vulnerabilities.
